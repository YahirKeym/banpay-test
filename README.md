## Description

Prueba para Banpay. Instrucciones de instalación e iniciar, probar y buildear la aplicación.

Solo se necesita colocar el .env correspondiente, deje un .env example.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
